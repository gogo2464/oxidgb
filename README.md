Oxidgb
======

Yet another work-in-progress Gameboy emulator for Rust.

Features
--------

- Broken code
- ~~No~~ audio emulation
- Crashes

Running
-------

Windows/Linux/macOS, assuming you have `cargo` on your PATH:

```bash
cargo run --release
```
